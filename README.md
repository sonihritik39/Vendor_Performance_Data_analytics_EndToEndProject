This project focuses on analyzing vendor performance using data analytics techniques to identify top-performing vendors, low-performing vendors, and overall procurement efficiency. The goal is to help businesses optimize their vendor relationships and improve decision-making using data-driven insights.

The project includes data cleaning, transformation, exploratory data analysis (EDA), visualization, and dashboard creation using Power BI.

Objectives:-

* Identify top and low-performing vendors
* Analyze brand-wise performance
* Evaluate purchase contribution of vendors
* Detect inefficiencies in procurement
* Build interactive dashboards for decision-making
